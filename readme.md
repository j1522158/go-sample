#

## create folder
```
mkdir go-sample
```

## create go.mod
```
go mod init go-sample
```

## run main.go
```
go run main.go
```

## build & run
```
go build -o app main.go
```
-o : BinaryFileName

```
./app
```

## external package install
```
go mod tidy
```
